Macroeconomic Stress Testing Framework for Consumer Credit Portfolios

Objective

This project studies how changes in macroeconomic conditions can affect consumer credit portfolios and expected credit losses.

Key Components

  • Probability of Default (PD)
  • Loss Given Default (LGD)
  • Exposure at Default (EAD)
  • Expected Credit Loss (ECL)

Stress Scenarios

Scenarios may include changes in unemployment, inflation, GDP growth, and interest rates.

The objective is to create a framework that helps quantify how portfolio credit risk changes under adverse economic conditions.
